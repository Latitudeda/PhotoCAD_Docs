PhotoCAD产品手册资料
====================================
**PhotoCAD** 采用了光子行业广泛使用的Python语言编写。开发人员对生成布局所使用的算法进行了精心优化和研究，使得 **PhotoCAD** 在国内外类似工具中具有独特优势，例如，版图在任何角度都不会出现扭曲或撕裂的问题。 **fnpcell** 和 **link** 是 **PhotoCAD** 的两个主要工具，它们对应于光子芯片布局设计中的参数化单元布局和链路级布局。

对于想要使用 **PhotoCAD** 创建光集成电路（PIC）的初学者，我们建议浏览 ``教程`` 部分，您可以从创建PCells到构建自己的电路，快速入门。

在 ``案例`` 部分还有由 **Latitudeds** 创建的多个应用程序，用户可以浏览这些示例，了解 **PhotoCAD** 如何实现您的创意。
