gpdk (generic PDK)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**gpdk** 是一个软件包，用于在 PhotoCAD 平台上创建基于 Python 脚本的参数化器件布局单元.

将 ``photocad-x.x.x-cp38-cp38-win_amd64.whl`` 文件放到 ``.python>.pypi_cache`` 文件夹中，即可安装或者升级gpdk软件包。

安装后，您可以在当前项目的 ``External Libraries`` > ``Python3.8 (PhotoCAD)`` > ``site-packages`` > ``gpdk folder`` 中找到 gpdk 包的完整内容，其中包含几个子文件夹: ``benchmark``, ``components``, ``examples``, ``routing``, ``simulation``, ``technology``, and ``util``.

这些文件存储在以下位置: ``D:\Softwareu\photocadu\PhotoCAD\.venv\Lib\site-packages\gpdk``, 其中: ``D:\Softwareu\photocadu\PhotoCAD`` 是项目路径 (用户安装 **PhotoCAD** 的路径)。
